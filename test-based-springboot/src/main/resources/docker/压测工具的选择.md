# 压测工具的选择

## 要求

1. 压测数据模拟，压测时能否模拟大量请求数据
2. 压测力度，单机压测上限、能否分布式等
3. 压测结果采集，如响应时间分布图和平均值、qps分布图和平均值、请求成功失败数分布图和总数、被压测机器的CPU、内存、磁盘、网卡数据
4. 使用难度
